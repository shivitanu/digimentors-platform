# DigiMentors

DigiMentors — Laravel 12 learning platform scaffold.

Setup

1. Copy `.env.example` to `.env` and set DB and Razorpay keys.
2. Run `composer install` and `npm install`.
3. Run `php artisan key:generate`.
4. Run migrations: `php artisan migrate --seed`.
5. Build assets: `npm run dev`.

This scaffold includes:
- TailwindCSS with dark mode
- Models: User, Course, Lesson, Resource, Blog, Payment
- Razorpay keys placeholders
- Basic Blade templates and routes
